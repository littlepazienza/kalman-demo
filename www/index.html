This is the Kalman Demo

Kalman moves in 2D space using a simple rotation and movement plc. The rotation plc controls the angle of movement, and the movement plc controls the speed of movement.


Goal

Use the inputs to set the x and y position of the goal





Debug Info

Vectors are positional information: x, y, velocity (pixels per ms), rotation (rad)
